# C Programming Notes

This repo is all my notes as I learn C. This will be a record
of my progress so future me can see that even though I have 
tried many times to learn C, I finally made it and on my way
to create great things that make me proud.

This repo will span through multiple resources and I plan to
keep content organized by having content for specific resources
in their own directories.

This will really show progress from knowing very little about
programming to eventually writing my own operating system. I
do believe I have a bit of Terry Davis energy at times when it
comes to programming and creating my own spin on software if
only my ADD would let me focus on something without going down
a rabbit hole in multiple directions.

- Programs that I written while reading book
